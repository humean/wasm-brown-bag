### Prerequisites

* [Node >= v8](https://nodejs.org/en/)

### Instructions

```sh
npm install

npm run start
```

* Open [http://localhost:8080/](http://localhost:8080/) is a recent version of firefox or chrome

* Open Chrome or Firefox Dev tools (typically F12) and click on "console"
